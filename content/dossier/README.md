# Abroad Study Research Dossier

Research-backed master’s and PhD application workspace for `Spring/Fall 2027`, built around this profile:

- `IIT Madras B.S. in Data Science and Applications`
- Expected completion: `Apr/May 2026`
- Current CGPA: `~7.0/10` (working estimate)
- Backend / GenAI engineering experience at `FreshMenu`

Last major review: `2026-04-08`

## Goal

Build a realistic, scholarship-aware path to study abroad without defaulting to high-cost self-funded applications.

This repo is optimized for:

- country prioritization
- university and programme shortlist building
- scholarship tracking
- common-document preparation
- fast execution once the live application cycles open

## Best Strategy

### Tier A
- `Germany`: strongest budget-to-outcome option
- `Taiwan`: strong technical ecosystem + better scholarship/value balance
- `Finland`: high-quality option if tuition waiver lands
- `South Korea`: very good funded AI / CS route
- `New Zealand`: especially strong for research-heavy and PhD routes

### Tier B
- `Singapore`: excellent but selective and expensive for coursework
- `UK`: efficient 1-year master’s route if funding is meaningful
- `Australia`: mainly useful for funded research or partial-fee awards

### Tier C
- `US`: use for funded PhD or thesis-MS only
- `Norway`: backup only for this budget profile

## Recommended Execution Order

### Phase 1: Preparation
- Build admissions CV
- Prepare IELTS / TOEFL
- Collect transcripts, passport, and proof documents
- Lock recommenders

### Phase 2: Shortlist
- Start with Tier A countries
- Use fit labels and funding fit labels in university files
- Ignore weak ROI options unless a funding path appears

### Phase 3: Application
- Recheck active cycle, fees, and current requirements from official portals
- Customize SOP per programme
- Submit scholarship-linked applications first

## Important Caveats

- This repo is a planning system, not an official admissions portal.
- Final eligibility always depends on the live university cycle and exact transcript mapping.
- Many `Fall 2026` windows were already closed when this dossier was created, so the main working target is `Spring/Fall 2027`.
- Scholarship availability can change every cycle; verify each funding link again before applying.

## Repo Purpose

This is not a generic study-abroad notes dump. It is a structured action repo for choosing affordable, high-upside graduate pathways in data science, AI, and adjacent computing fields.
